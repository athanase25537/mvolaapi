Environnement Sandbox

Vous pouvez utiliser les identifiants de test fournis par MVola pour simuler :

1. Paiements simulés

  • Utilisez des numéros de téléphone fictifs comme 0343500000
  • Testez différentes valeurs de montant (min, max, erreur)

2. Observation des réponses

Examinez les codes HTTP et les objets JSON retournés.

3. Suivi de transaction

Appelez /transaction/:id pour voir l’état de la transaction simulée.

4. Restrictions de sandbox

  • Les transactions ne sont pas réelles
  • Pas de transfert d’argent
  • Limité à certains endpoints
