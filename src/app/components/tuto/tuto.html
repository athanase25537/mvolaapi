Tutoriels API MVola

Tutoriel 1 : Créer un compte développeur MVola

  1. Accéder au portail développeur MVola : developer.mvola.mg
  2. S'inscrire et valider votre e-mail
  3. Créer une application pour obtenir Client ID et Client Secret

Tutoriel 2 : Générer un access token

  1. Utilisez Postman ou curl pour faire une requête POST à /token
  2. Inclure les bons headers et body
  3. Récupérer le token pour les appels suivants

Tutoriel 3 : Envoyer un paiement test

  1. Préparer un body JSON avec amount, debitParty, etc.
  2. Appeler l’endpoint `/payment` avec le token
  3. Recevoir une réponse contenant le statut de la transaction

Tutoriel 4 : Gérer les erreurs

  • Ajouter des validations côté frontend
  • Afficher un message clair à l'utilisateur selon le code erreur
